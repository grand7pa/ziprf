# PDF lead magnet viewer

Мини-сайт для просмотра PDF-презентации через GitHub Pages.

## Как запустить

1. Создай новый репозиторий на GitHub.
2. Загрузи в репозиторий:
   - `index.html`
   - `presentation.pdf`
   - `.nojekyll`
3. Открой Settings → Pages.
4. Включи публикацию из ветки `main`, папка `/root`.
5. Ссылка будет вида:

```text
https://USERNAME.github.io/REPOSITORY/
```

## Как заменить PDF

Самый простой вариант:

- Удали старый `presentation.pdf`.
- Загрузи новый PDF с таким же названием: `presentation.pdf`.

Или можно хранить несколько PDF и открывать через параметр:

```text
https://USERNAME.github.io/REPOSITORY/?file=franchise.pdf
```

Файл `franchise.pdf` должен лежать в этом же репозитории рядом с `index.html`.

## Как подключить к Тильде

В форме на Тильде укажи Success URL / Thank You Page URL:

```text
https://USERNAME.github.io/REPOSITORY/
```

После отправки формы человек попадёт на PDF-просмотрщик.

## Важно

Это не защищённое хранилище. Если человеку известна ссылка, он сможет открыть презентацию без повторной заявки.
Для обычного лид-магнита это нормально. Для закрытых материалов нужен другой вариант: отправка персональной ссылки на почту/в CRM или закрытый доступ.
